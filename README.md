# OGA™ Roadmap

Public roadmap for the OGA ecosystem — [roadmap.oneearthrising.com](https://roadmap.oneearthrising.com)

## Stack
- Vite + React 18
- Supabase (data layer)
- Netlify (hosting)

## Development
```bash
npm install
npm run dev     # http://localhost:5174
```

## Deploy
Pushes to `main` auto-deploy via Netlify.

## Access Tiers
- **Public** — default, no auth
- **Investor** — passcode via footer lock icon
- **Internal** — OGA admin auth (planned)
